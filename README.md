# Practica I: Perfiles de usuario

# # Información del Proyecto

Este repositorio contiene un programa Java que permite administrar perfiles de usuarios. La aplicación permite crear nuevos usuarios con información como nombre completo, correo electrónico y código de perfil. Luego, se pueden mostrar los perfiles de usuarios existentes.

## Integrante

- **Nombre Completo del Integrante:** Manuel Aristizabal Arbelaez

## Versiones de Compilador e IDE

- **Versión del Compilador:** javac    17.0.3
- **IDE Utilizada:** IntelliJ IDEA

## Descripción del Proyecto

Este proyecto es una aplicación de línea de comandos escrita en Java que permite realizar las siguientes acciones:

1. Crear nuevos usuarios con información como nombre completo, correo electrónico y código de perfil.
2. Mostrar los perfiles de usuarios existentes.

## Instrucciones de Uso

Para ejecutar el programa, sigue estos pasos:

1. Clona este repositorio en tu entorno de desarrollo.

2. Abre el proyecto en tu IDE (en este caso, se utilizó IntelliJ IDEA).

3. Ejecuta la clase `Main` para iniciar la aplicación.

4. Selecciona una de las siguientes opciones:
    - **1. Crear nuevo usuario:** Ingresa la información del nuevo usuario cuando se te solicite.
    - **2. Mostrar perfiles de usuarios:** Verás los perfiles de usuarios existentes.

5. Sigue las instrucciones proporcionadas por la aplicación para crear usuarios o mostrar perfiles.

## Notas Adicionales

- El proyecto incluye una clase `Usuario` que representa a los usuarios y muestra información adicional según el código de perfil.
- Asegúrate de proporcionar la versión correcta del compilador de Java al compilar y ejecutar el programa.

¡Disfruta explorando y utilizando esta aplicación de administración de usuarios en Java!
